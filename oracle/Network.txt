용어정리

프로토콜 : 컴퓨터 내부 혹은 컴퓨터 사이에서 데이터의 교환 방식을 정의하는 규칙 체계

OSI참조모델 : 프로토콜을 7계층으로 분류
				상	위	층
	응용 계층 : 사용자와 직접 상호작용하며 인터페이스 역할을 수행하고, 각 프로세스 간의 정보 교환을 담당한다.
			※ 주요 프로토콜 : HTTP, FTP, DNS, DHCP
	표현 계층 : 운영체제를 통해 입출력되는 데이터를 다른 표현 형태로 변환한다.
		     	※ 주요 프로토콜 : JPEG, MPEG, ASCII
	세션 계층 : 응용 프로세스를 통해 통신 세션을 구성한다.
		    	※ 주요 프로토콜 : TLS, SSH

				하	위	층
	전송 계층 : 종단에 신뢰성 있고 에러 복구와 흐름 제어, 다중화 기능을 담당
		    	※ 데이터 전송 단위: 세크먼트(Segment)
			※ 주요 프로토콜 : TCP, UDP
    네트워크 계층 : 다중 네트워크 링크에서 패킷을 발신지로부터 목적지까지 성공적으로 효과적으로 전달한다.
		    	※ 데이터 전송 단위: 패킷(Packet)
			※ 주요 프로토콜 : IP, ICMP, IGMP, ARP
  데이터링크 계층 : 인접 장치간의 정보 전송 동기화, 에러제어, 흐름 제어 기능을 담당한다.
		    	※ 데이터 전송 단위: 프레임(Frame)
			※ 주요 프로토콜 : CSMA/CD, LLC, Token Ring
	물리 계층 : 물리적 매체를 통해 0과 1의 비트 흐름을 전송하기 위해 요구되는 기능들을 조정한다.
		    	
TCP/IP모델  : 프로토콜을 4계층으로 분류
	응용 계층 : (OSI 상위층에 해당)
			응용프로그램 간의 데이터 송수신
	전송 계층 : (OSI 전송 계층에 해당)
			호스트 간의 자료 송수신
      인터넷 계층 : (OSI 네트워크 계층에 해당) 
			데이터 전송을 위한 논리적주소 지정 및 경로 지정
  네트워크 엑세스 : (OSI 데이터 링크, 물리 계층에 해당)
			실제 데이터인 프레임을 송수신
LAN : 근거리 통신망, 로컬 영역 네트워크(Local Area Network)
	네트워크 메체를 이용하여 집, 사무실, 학교 등의 건물과 같은 가까운 지역을 한데 묶는 컴퓨터 네트워크이다.

WAN : 광역 통신망 (Wide Area Network)
	넓은 지리적 거리/장소를 넘나드는 통신/컴퓨터 네트워크

MAC : (Media Access Control)  
	네트워크 세그먼트의 데이터 링크 계층에서 통신을 위한 네트워크 인터페이스에 할당된 고유 식별자이다.

NIC : (Network Interface Controller) 네트워크 어댑터
	컴퓨터를 네트워크에 연결하여 통신하기 위해 사용하는 하드웨어 장치

이더넷 : (Ethenet) 컴퓨터 네트워크 기술의 하나
	OSI모델의 물리 계층에서 신호와 배선 데이터 링크 계층에서 MAC 패킷과 프로토콜의 형식을 정의한다.

패킷 : (Packet) 네트워크 전송의 용량 단위
	
IP : (Internet Protocol) 송신 호스트와 수신 호스트가 패킷 교환 네트워크에서 정보를 주고받는 데 사용하는 정보 위주의 규약

IP주소의 특징 : 사설주소의 범위
	비신뢰성, 비연결성

라우터 : 패킷을 최종 목적지까지 빠르고 안전하고 전송하는 과정인 라우팅을 수행하는 네트워크 장치

TCP : (Transmission Control Protocol) 전송 제어 프로토콜
	IP와 함께 TCP/IP라 불리며 통신망에 연결된 컴퓨터에서 실행되는 프로그램 간에 안정적으로 순서대로 에러없이 교환할 수 있게 한다.
http : (Hyper Text Transfor Protocol) 
	웹상에서 정보를 주고 받을 수 있는 프로토콜
https : (http + Secure) 암호화된 버전
	클라이언트와 서버 간의 모든 커뮤니케이션을 암호화 하여 민감한 정보를 안전하게 주고 받도록 해준다.

DNS : (Domain Name System) 호스트의 도메인 이름을 네트워크 주소로 변경하거나 그 반대로 수행할 수 있게 변환해준다.

DHCP : (Dynamic Host Configuration Protocol) 호스트 IP 구성 관리를 단순화하는 IP 표준
